1. Project Overview
The goal of the project is to create a functional to-do list where users can:

Add new tasks
Mark tasks as completed
Delete tasks
Filter tasks (All, Completed, Uncompleted)

2. Technologies Used
HTML: For structuring the layout and elements of the to-do list (buttons, task list, input fields).
CSS: To style the to-do list, making it visually appealing (color, spacing, fonts).
JavaScript: To add interactivity, such as adding tasks, marking them as completed, and deleting tasks.
